# ST Chat Jumper

SillyTavern 悬浮跳转条插件，提供快捷楼层导航、区间显示、快速编辑、快捷翻页、临时收藏等功能。

## 功能

<video src="https://github.com/user-attachments/assets/04cab1ea-01f4-4179-a2bd-625b945f051a.mp4" controls="controls" width="80%" loop></video>

- **悬浮可拖拽** — 跳转条可自由拖拽到屏幕任意位置，支持横/竖布局切换
- **快速跳转** — 最近 3 楼 / 快速翻页（左/右）/ 上一楼 / 下一楼（跳到头部）
- **精确对齐** — 当前楼层头部 (H) / 尾部 (L) 对齐
- **楼层区间显示** — 输入区间仅显示指定范围的楼层，可随时恢复默认视图
- **快速编辑** — 选中文字可高亮定位到编辑器对应位置；未选中则编辑当前楼层
- **快速翻页** - 可以快速在最新楼层里左右翻页/重roll
- **临时收藏** — 点选收藏楼层，快速跳转到收藏的楼层（仅当前页面有效）
- **按钮自定义** — 在扩展设置面板中自定义按钮的显示/隐藏、排列顺序、悬浮栏大小

## 按钮自定义（v1.5.0 新增）

在 SillyTavern 的 **扩展设置** 中，展开 **ST Chat Jumper** 面板：
- **勾选/取消** — 控制每个按钮的显示或隐藏
- **拖拽排序** — 拖动左侧手柄 `⠿` 调整按钮在悬浮条中的排列顺序
- 设置会自动保存，刷新后保持

## 安装

- 自动安装
打开 SillyTavern → 扩展 → 安装扩展
输入以下url：
```
https://github.com/qianzhuowo/ST-Chat-Jumper
```

- 手动安装
把整个`ST Chat Jumper`文件夹放到：
```text
SillyTavern/public/scripts/extensions/third-party/
```
然后在 SillyTavern 的扩展管理里启用该插件。


### 可配置按钮

| 设置项 | 说明 |
|--------|------|
| 收起/展开 | 悬浮条的折叠/展开切换按钮 |
| 最近第3楼 | 跳转到倒数第 3 条消息 |
| 最近第2楼 | 跳转到倒数第 2 条消息 |
| 最近第1楼 | 跳转到最后一条消息 |
| 快速翻页(右) | 最新楼层的右翻页/重roll |
| 快速翻页(左) | 最新楼层的左翻页 |
| 显示楼层区间 | 区间显示/恢复/指示器（三个元素为一组） |
| 横/竖布局切换 | 切换悬浮条的横向或纵向排列 |
| 上一楼 | 跳转到上一楼（头部对齐） |
| 下一楼 | 跳转到下一楼（头部对齐） |
| 对齐到头部 (H) | 将当前楼层对齐到视口顶部 |
| 对齐到尾部 (L) | 将当前楼层对齐到视口底部 |
| 快速编辑 (✏️) | 打开/保存快速编辑 |
| 收藏 (📌) | 收藏楼层功能组（图钉 + 展开箭头） |
